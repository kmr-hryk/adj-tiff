adj-tiff
====

## これは何

ファイル名が地番名となっている地積測量図を、その地番名のディレクトリを作成してその中に「sokuryouzu##.tif」というファイル名に変更して移動します。

例えば

* 2-13.tif
* 2-14(1-2).tif
* 2-14(2-2).tif
* 2-16.tif

という4つのTIFFファイルがあった場合。

* 2-13
 - sokuryouzu01.tif
* 2-14
  - sokuryouzu01.tif
  - sokuryouzu02.tif
* 2-16
  - sokuryouzu01.tif

という状態になります。

とりあえずMITライセンスで運用してください。

## 使い方

PowerShellスクリプトです。セキュリティ設定で署名無しのスクリプトを実行可能なように設定をしておいてください。

適当なディレクトリにadj-tiff.ps1を置き、ファイル名を変更したいディレクトリに移動後、adj-tiff.ps1を実行してください。引数などは不要です。
